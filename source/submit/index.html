Levels and Mods for Jedi Knight / Mysteries of the Sith

All levels and mods should come with the required readme file, as stipulated by LucasArts. Here's a sample you can download, modify and use: sample_readme.txt.

Every submission email should include 3 attachments:

  • A zip file containing
    • The level/modification in GOB format
    • The accompanying readme file
  • At least two screenshots of your level or mod in action

In the body of the email, please provide:

  • The type of file you're submitting (JK/MotS Single- or Multiplayer Level or Modification)
  • The name of the level/mod
  • The author's name
  • The author's e-mail address (tell us if you don't want it posted directly on the site)
  • A nice description of the level or the purpose of the mod

Here is a small example for your viewing pleasure:

Subject: Level/Mod Submission for Massassi

Type: JK Multiplayer Level
Name: 5600 Block
Author: Brian Lozier

Description:
This is a small multiplayer level set on a remote and abandoned planet. It consists of 4 main buildings surrounding a pool of water. There is a tower coming up from the pool that is designed for saber dueling. This level is intended for 2-4 players. Being so small, this level should provide for some fast and furious gameplay.

Email it all. Address up towards the top.

Maps and Modifications for Jedi Outcast

All maps and mods should be submitted in PK3 format, and they should also come with the required readme file, as stipulated by LucasArts. Here's a sample readme file you can download and use: sample_readme_jk2.txt.

Every submission should have the following:

  • The level/modification in PK3 format
  • The accompanying readme file with all the necessary information
  • At least two screenshots of your level or mod in action

Place everything listed above in a ZIP file and attach it to an e-mail whose body should have all the following information:

  • The type of file you're submitting (JO SP, FFA, CTF, Jedi Duel, or Modification)
  • The name of the map/mod
  • The author's name
  • The author's e-mail address (tell us if you don't want it posted directly on the site)
  • A proper description of the level or the purpose of the mod

Email it all. Address up towards the top.

Skins for Jedi Outcast

Skin submissions should be sent in ZIP format and should include all of the following:

  • All the skin-related files in a PK3 file
  • A readme file with a description and the necessary info (download a sample file here)
  • Two screenshots of your skin in action

Screenshots should be in JPG format. If you're going to use someone else's skin as the base for yours, please get their permission first.

Email it all. Address up towards the top.
