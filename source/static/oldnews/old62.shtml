New Version of JkEdit 3.05! - darkjedi86
Yes, a new version of JkEdit has been released, which is available at the JkEdit Website! Here's the new enhancements and fixes:
  • A new command for grid nodes on surfaces only
  • A new command for grid nodes on edges only
  • A new option for Sound Selection for a set of sectors
  • A new option for transparent rendering of adjoined surfaces
  • The Create Light command has been improved
  • Expanding to opposite surface having zero distance to current surface is now allowed
  • Merging of surfaces having adjoined surfaces are now allowed
  • Merging of sectors with adjoined surfaces now works correctly
  • Texture alignment using copy/paste now works correctly for different length of surface edges
  • Splitting of a surface in any split/expand operation has been improved to use a rectangular extent
  • Usage of the JK/Mots resource file has been improved to eliminate "Cannot find resource file"
  • Multiple transparent surfaces are now rendered in correct order
    • Woah, that's a lot of new features. So go pay the JkEdit Website a visit today and check it out!
