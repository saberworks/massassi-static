The SWTTC preview has finally hit the information superhighway. Now, before you jump quickly to the Files section to get it, let me make a few things clear. THIS IS NOT THE FULL LEVEL!! New Versions of JkEdit - Brian
The JkEdit Homepage has been updated with new versions of JkEdit.
Versions 2.50/1.50 of JkEdit contain some new features:
  • DirectX Immediate mode used instead of Retained mode
  • Improved selection of surfaces when "outside"
  • Improved performance of changing Options parameters

Versions 2.50/1.50 of JkEdit contains a few bug fixes:
  • Some sounds could not be played
By using DirectX Immediate mode, these versions of JkEdit have highly increased performance, achieved through a 4MB PCI or AGP 3D Graphics Accelerator or better.
